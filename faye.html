王菲

基本信息

  • 王菲（Faye Wong），1969年8月8日出生于北京，中国流行乐女歌手、影视演员 ，中国国家一级演员 。 1989年，凭借粤语歌曲《仍是旧句子》在香港乐坛出道；同年发行首张个人专辑《王靖雯》 。1992年，因演唱歌曲《容易受伤的女人》而被听众所熟知。1993年，在粤语专辑《十万个为什么？》 中尝试另类音乐的风格 。1994年，发行普通话专辑《天空》，并获得十大劲歌金曲最受欢迎女歌星奖；同年在香港红磡体育馆举办18场“最精彩演唱会”，打破中国香港歌手初次开演唱会的场次纪录。
  • 1996年，成为首位登上美国《时代周刊》封面的华人歌手 。1998年，因在央视春晚上演唱歌曲《相约一九九八》而在中国内地获得更多的关注 ；同年发行专辑《唱游》，通过运用颤音、拖音等技巧确立其个人风格，并于次年获得新加坡SPVA唱片畅销排行榜年度销量冠军 。
  • 1999年，凭借英文歌曲《Eyes on Me》获得第41届日本唱片大奖亚洲音乐奖 ，同年在日本武道馆举行个人演唱会 。2002年，获得吉尼斯世界纪录大全颁发的“唱片销量最高的粤语女歌手”证书 。2003年主演喜剧电影《天下无双》。2004年，凭借专辑《将爱》获得第15届台湾金曲奖最佳国语女演唱人奖 。2005年，在结束“菲比寻常巡回演唱会”后淡出乐坛 。2010年，复出乐坛并在央视春晚上演唱歌曲《传奇》，同年举办巡回演唱会 。2018年，与那英在春晚演唱《岁月》 。
  • 演艺事业外，王菲热心公益慈善。2006年11月，成立嫣然基金会 。2012年5月，成为北京嫣然天使儿童医院的创始人之一 。

早年经历

  • 王菲小时候与父母、哥哥一起居住在北京青年沟的煤矿大院 。1975年，王菲就读于北京地坛小学，并担任班上的文艺委员。
  • 1980年，王菲就读于北京市东直门中学 。在读中学期间，她加入了中央电视台银河少儿艺术团。1983年，她在央视的六一晚会上独唱《大海啊故乡》《小百灵之歌》等歌曲。1984年，她在东直门中学读高中。高中二年级时，她出版了翻唱邓丽君歌曲的作品《风从那里来》、《邓丽君故乡情》等等 。
  • 1987年，王菲放弃厦门大学生物系的录取资格并跟随父亲移居香港；同年，她经朋友介绍拜戴思聪为师，并在他的指导下学习声乐 。

演艺经历

新艺宝唱片

(1989~1966年)
  1. 1989年，王菲凭借粤语歌曲《仍是旧句子》并凭借该歌曲获得亚太金铮流行曲创作大赛铜奖 ，之后在香港歌坛出道 ；同年，签约新艺宝唱片公司；10月，她以“王靖雯”为艺名并推出首张个人专辑《王靖雯》 ；该专辑发行后达到金唱片的销量 ，而王菲亦凭借这张专辑获得叱咤乐坛流行榜叱咤乐坛女歌手铜奖。1990年5月，发行专辑《Everything》 ，该专辑也奠定了王菲的歌路 ；8月，推出专辑《You're The Only One》，并逐渐形成个人化的音乐路线。
  2. 1991年，发行精选专辑《Shirley Once More》；同年，赴美国修读音乐方面的课程 ；此外，她还为无线拍摄了电视电影《别姬》，在片中饰演思想单纯的的歌女阿芳 。1992年，从美国返回香港；7月，发行融合中西方音乐风格的粤语专辑《Coming Home》，并通过该专辑开始形成拖音唱腔 ；专辑中的歌曲《容易受伤的女人》在推出后获得关注，并获得十大劲歌金曲颁奖礼十大金曲奖、十大中文金曲颁奖礼十大金曲奖、叱咤乐坛流行榜至尊歌曲等奖项。
  3. 1993年2月5日，王菲发行了粤语专辑《执迷不悔》，她在专辑中转变音乐风格，并首次加入摇滚乐 ；专辑主打歌《执迷不悔》是她首次尝试歌词创作的作品，并获得大中文金曲奖、十大劲歌金曲奖等奖项；3月1日，参演的时装科幻剧《原振侠》播出，并演唱该剧的插曲《软弱》；4月，主演时装律政剧《壹号皇庭II》，在剧中饰演办事能干的检察官唐毓文；9月，在返回香港后发行了粤语专辑《十万个为什么？》，她在专辑中将独立流行乐和个人声线合而为一，尝试另类音乐的风格 ；在这张专辑中，王菲参与了包括词曲创作、造型设计、封面概念设计在内的幕后工作，而其中的歌曲《动心》是王菲第一次包揽作词和作曲的作品 ；12月，发行个人首张EP《如风》；同年，获得十大劲歌金曲国内最受欢迎香港女歌星奖、叱咤乐坛流行榜叱咤乐坛女歌手金奖等奖项。
  4. 1994年4月，发行个人首张普通话专辑《迷》，主打歌《我愿意》获得第1届华语音乐榜中榜ChannelV中文Top20榜中榜歌曲奖 ；6月，王菲不再使用艺名“王靖雯”，而以本名“王菲”为名并推出粤语专辑《胡思乱想》，她在专辑中翻唱了多首英伦另类音乐风格的歌曲，其中《梦中人》是她首度运用咽音技巧的作品 ；7月14日，王菲主演的文艺片《重庆森林》上映，她在片中饰演具有独立想法的速食店打工女孩阿菲；11月，发行普通话专辑《天空》，其中《天使》是其首度尝试长线卷音吟唱的作品 ，而主打歌《棋子》则获得第3届新加坡金曲奖最佳金曲奖；该专辑发行第二周便获得台湾金曲龙虎榜国语销量榜冠军，并在金曲龙虎榜销售榜前10名的位置停留23周，专辑在台湾的年度销量近80万张，在台湾的累计销售点数为78660点，在亚洲的累计销量突破300万张；至此，王菲凭借《胡思乱想》《天空》两张专辑形成个人风格，她凭此横扫了1994年中国香港乐坛四大颁奖礼的所有女歌手奖：包括十大劲歌金曲的亚太区最受欢迎女歌手、最受欢迎女歌手；叱咤乐坛女歌手金奖；十大中文金曲全球华人最受欢迎女歌手以及新城劲爆女歌手奖 ；11月起，在温哥华、多伦多、亚特兰大等地举行演唱会 ；12月，发行粤语专辑《讨好自己》，并为专辑创作了主打歌《讨好自己》以及RAP说唱曲《出路》；该专辑整体凸显英伦风格，在香港IFPI唱片销量榜前十名的位置停留10周，在香港地区的累计销量突破三白金；12月22日起，在香港红磡体育馆一连举办18场演唱会，打破中国香港歌手初次开演唱会的场次纪录。
  5. 1995年，凭借文艺片《重庆森林》获得第14届香港电影金像奖最佳女主角提名 ；5月，发行EP《一人分饰两角》；7月，发行向邓丽君致敬的翻唱专辑《菲靡靡之音》；12月，推出英伦迷幻风格的粤语专辑《Di-Dar》，并为专辑创作了《Di-Dar》和《假期》两首歌曲 ；专辑发行第二周获得香港IFPI唱片销量榜冠军，并在榜单前十名的位置停留10周，在香港的累计销量突破三白金，专辑主打歌曲《暧昧》获得第19届十大中文金曲颁奖礼最佳改编歌曲奖；同年，获得第3届新加坡金曲奖最受欢迎女歌手奖、十大劲歌金曲颁奖礼亚太地区最受欢迎女歌手等奖项。
  6. 1996年5月，发行以个人自主状态作为主导概念的专辑《浮躁》；该专辑不同于主流流行音乐的风格，王菲将Brit Pop及部分Dream Pop元素加入到专辑中 ，并为专辑创作了《无常》、《堕落》等8首歌曲 ；王菲也凭借该专辑在国际上获得关注，并成为美国《时代周刊》的封面人物 ；同年，获得第7届台湾金曲奖最佳国语女演唱人提名、十大劲歌金曲颁奖礼亚太区最受欢迎女歌手等奖项。

百代唱片

(1997~2001年)
  1. 1997年2月，发行粤语EP《玩具》；同年，以6000万港币的签约费与百代唱片签约，并宣布不再出粤语专辑；9月，推出普通话专辑《王菲》，并融合英式摇滚等曲风；该专辑发行后在台湾的销量超过50万张 ，其中主打歌《你快乐所以我快乐》获得十大中文金曲颁奖礼全球华语金曲奖、全球华语音乐榜中榜ChannelV中文Top20榜中榜歌曲奖 ；10月，发行粤语EP《自便》；同年，推岀精选集《菲卖品》 。
  2. 1998年春节，王菲与那英在央视春晚上合唱歌曲《相约1998》，并获得第1届CCTV-MTV音乐盛典内地最受欢迎流行歌曲奖 ；9月，推出专辑《唱游》，并首度尝试晒伤妆造型 ；除了担任专辑监制并创作《感情生活》、《脸》等4首歌曲外 ，她还在专辑中运用美声、颤音、拖音、假声、气声等演唱技巧 ；专辑发行后在亚洲的累积销量超过250万张 ，并蝉联马来西亚RIM中文唱片销售榜7周冠军；而在新加坡，该专辑在SPVA销量榜前十名的位置停留6个月的时间，并于次年获得新加坡SPVA唱片畅销排行榜年度销量冠军 ；10月起，在广州、深圳、昆明等地举行“唱游大世界演唱会” 。
  3. 1999年2月，演唱电玩主题曲《Eyes on Me》 ，该曲在日本发行后位列日本Oricon销量榜第9位，使得王菲成为首位凭借非日文歌曲而进入Oricon榜单前10位的华人歌手 ；该曲的同名专辑在日本的销量超过35万，成为日本外语唱片的销量冠军 ，并获得第41届日本唱片大奖亚洲音乐奖 ；3月，在日本武道馆举行个人演唱会 ；8月21日，在吉隆坡武吉加利尔室内体育馆举行个人演唱会 ；9月，发行普通话专辑《只爱陌生人》，除了为专辑创作歌曲《精彩》外，她还首度尝试演唱拉丁曲风的歌曲 ；该专辑年度销量超过80万张 ，并于次年获得新加坡SPVA唱片畅销排行榜年度销量冠军以及第7届新加坡金曲奖年度最畅销专辑奖；同年，获得十大中文金曲颁奖礼全球华人最受欢迎女歌手、十大劲歌金曲颁奖礼亚太区最受欢迎女歌手等奖项。
  4. 2000年，凭借专辑《只爱陌生人》获得第11届台湾金曲奖最佳国语女演唱人奖提名；7月，主演爱情片《恋战冲绳》，在片中饰演个性格率直的酒吧女老板Doris ；10月，发行普通话专辑《寓言》，并为专辑创作了电子风格的《新房客》、Trip-Hop风格的《彼岸花》等5首歌曲 ，该专辑的风格回归主流音乐 ，并获得中华音乐人交流协会年度十大专辑奖、第1届全球华语歌曲排行榜最佳专辑等奖项；同年，在台北、南京、长沙、墨尔本、悉尼等地举行世界巡回演唱会 。
  5. 2001年3月，获得第12届台湾金曲奖最佳国语女演唱人奖提名 ；4月，主演日本励志喜剧《弄假成真》，在剧中饰演积极朝目标迈进的留日学生林菲 ；9月7日至8日，在广州体育馆举行演唱会，并演唱了《我愿意》、《人间》等代表作 ；9月22日，在上海虹口体育场举行演唱会；10月，推出普通话专辑《王菲2001》 ，专辑中的歌曲《光之翼》在日本Oricon榜单排名第19位 ，而专辑主打歌《流年》则获得第2届华语音乐年度大奖十大华语歌曲奖、第13届台湾金曲奖最佳录影带奖 ；7月25日，发行单曲《Separate Ways》 ，该曲是日剧《弄假成真》的主题曲，发行后在日本Oricon榜单排名第14位 ；11月1日至2日，在东京武道馆举办两场“全面体演唱会” 。

新力唱片

  1. 2002年，王菲与新力唱片公司签下合约；3月，发行精选专辑《你王菲所以我王菲》；5月，以970万的唱片总销量（统计时间截至到2000年3月）获得吉尼斯世界纪录大全颁发的“唱片销量最高的粤语女歌手”证书 [14-15] ；同年，与梁朝伟、赵薇共同主演喜剧片《天下无双》，在片中饰演女扮男装的无双公主，并凭借该片获得第22届香港电影金像奖最佳女主角奖提名以及第9届香港电影评论学会最佳女演员奖 。
  2. 2003年11月7日，推出普通话专辑《将爱》，除了担任唱片监制外，她还为专辑创作了摇滚风格的《将爱》、Trip-Hop风格的《不留》等四首歌曲 ；该专辑融合了抒情、电子乐、Trip- Hop等曲风，并获得IFPI唱片销量大奖十大销量国语唱片、华语音乐传媒大奖年度十大唱片奖等奖项 ；12月20日至12月27日，在香港红磡体育馆连开8场“菲比寻常演唱会”，并演唱了《执迷不悔》、《红豆》等代表作 ；同年，获得第4届中国金唱片奖港台女艺人奖、第5届CCTV-MTV音乐盛典亚洲地区最佳女歌手奖 。
  3. 2004年4月，凭借歌曲《不留》入围第15届台湾金曲奖最佳作词人奖 ；5月，凭借专辑《将爱》获得第15届台湾金曲奖最佳国语女演唱人 ；9月28日，王菲与梁朝伟、巩俐、木村拓哉共同主演的剧情片《2046》上映，她在片中饰演个性犹豫不定的王靖雯；同年，在吉隆坡、新加坡、杭州等地举行“菲比寻常演唱会” 。

淡出歌坛

  1. 2005年3月，位列福布斯中国名人榜第5位 ；5月底，王菲淡出歌坛 ；同年凭借文艺片《2046》获得第5届华语电影传媒大奖最佳女演员奖提名 。
  2. 2008年，王菲演唱的歌曲《天空》入选《南方周末》评出的“改革开放三十年之十大歌曲” 。2009年12月，发行EP《爱笑的天使》，主打歌《心经》是剧情片《唐山大地震》的片尾曲 。

复出歌坛

  1. 2010年1月，王菲正式复出歌坛，并为电影《孔子》演唱了主题曲《幽兰操》 ；2月13日，登上2010年中央电视台春节联欢晚会并演唱了歌曲《传奇》 ；同年10月起，先后在北京、上海各举行了5场“王菲巡回演唱会”，演唱会开票10天内的预售成绩达1888万元，打破中国内地演唱会门票的预售纪录 。 2011年1月，与陈奕迅共同演唱爱情片《将爱情进行到底》的主题曲《因为爱情》 ，并获得第4届华语金曲奖年度最佳国语歌曲奖、第8届中国金唱片奖数字音乐金曲奖等奖项 ；3月，在香港亚洲国际博览馆举行5场个人演唱会，并演唱了《约定》、《给自己的情书》等代表作；同年，在武汉、哈尔滨、成都等地举行巡回演唱会。
  2. 2012年1月，发行单曲《愿》；2月，在重庆奥体中心举行个人演唱会，演唱会以“重生与四季”为主题，展现了王菲的音乐之路 ；同年，获得第8届中国金唱片奖数字音乐类歌手奖。 2013年5月19日，王菲在“邓丽君60周年演唱会”上以歌声向偶像追思，她与邓丽君隔空对唱了其生前未公开的歌曲《清平调》 ；7月26日，为电影《致我们终将逝去的青春》演唱的主题曲《致青春》发布 ，该曲获得第9届中国金唱片奖金曲奖 ；同年，王菲开始处于半退隐状态 。 2014年11月8日，王菲演唱的电影同名主题曲《匆匆那年》发布，发行首日播放量达到627万，创下电影宣传视频的单日播放量纪录
  3. MV推出5天后播放量超过2000万 ；同年，为电影《触不可及》演唱主题曲《爱不可及》 。 2015年2月，在英国市场研究公司YouGov公布的调查中，王菲位列“全球最受仰慕的女性第17名” ；5月，在第5届全球流行音乐金榜颁奖礼中，凭借歌曲《匆匆那年》获得年度最佳电影歌曲奖、年度二十大金曲奖；9月，为电影《港囧》演唱主题曲《清风徐来》 ；11月，发行粤语专辑《敷衍》，这是王菲第十张粤语专辑 。 2016年7月22日，发行个人填词的单曲《尘埃》 ；同年，为电影《从你的全世界路过》演唱片尾曲《你在终点等我》
  4. 12月30日，在上海举办单场幻乐一场演唱会 ，该演唱会同步全球166个国家，通过360全景网络+VR全景进行全程直播，直播期间总观看人数超过2000万，累计总播放量达3.5亿，且“收割”了56万条弹幕 。2017年11月3日，王菲与马云共同为电影《功守道》演唱的主题曲《风清扬》发布 。 2018年1月9日，王菲演唱的电影同名推广曲《无问西东》发行 ；2月15日，登上2018年中央电视台春节联欢晚会，并与那英合唱歌曲《岁月》 ；7月20日，在国内首档音乐创演秀节目《幻乐之城》中担任“幻乐体验官”
  5. 12月9日，王菲加盟并担任“国宝守护人”的央视文博探索节目《国家宝藏第二季》播出，她在首期节目中以民国《大公报》记者的视角，讲述了民国时期“清样式雷建筑烫样”的故事 ；同年，王菲还为电影《邪不压正》演唱了推广曲《偶遇》 。 2019年，为电影演唱了同名主题曲《我和我的祖国》 。2020年1月19日，为电影《夺冠》演唱片尾曲《生命之河》 ；9月9日，王菲在全明星晚会“99划算盛典”中翻唱了《Landing Guy》《下雨天》《私人公园》等多首歌曲，还与马云隔空合唱《如果云知道》。
